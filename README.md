# Robot

Simple Robot which is doing user defined actions for you. <br>
You can save your Actions configs to persist your work.

# Environment
Works on:
  - Windows
  - Linux
  
Depends:
  - Java >= 1.8

# Usage

  java -jar Robot.jar 
  
  
# Config

Options category 'gui':

    --[no]gui [-g] (a boolean; default: "true")
      simple gui
